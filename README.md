# Help-ru

Для просмотра сайта перейдите на http://help-our-ru.herokuapp.com/

Для запуска сайта на локальном компьтере:
1. Разорхивируйте zip-архив
2. Выполните команду pip install -r requirements.txt в виртуальной среде проекта
3. Запустите main.py
4. Затем в другом терминале запустите команду celery -A main.client worker -l INFO -P gevent

----------------------------------------------------------------------------------------------------------------------------
Вы можете зарегестрироваться и задавать вопросы, а также отвечать на них, получая баллы, которые нужно для того чтобы задать вопрос.
Так же можно пожаловаться на ответы, которые вас не устраивают, и тогда система удалит их :)
Еще можно ставить оценку ответам от 1 до 5.
----------------------------------------------------------------------------------------------------------------------------


Структура:
• api – папка с api для проекта

• data – содержит модели бд и файлы запуска с подключением сессии

• forms – включает в себя все формы, которые есть на сайте

• migrations – миграции бд

• static – все статические файлы

• .flaskenv – содержит переменную окружения flask(для работы команд по типу flask dv init)

• Procfile – файл для загрузки heroku.

• celeryconfig – настройки celery

• main – основной файл со всеми обработчиками страниц

Задачи, решенные с помощью модуля celery:

• При отправлении жалобы пользователя на чужой ответ, программа через 60 секунд проверяет, что если кол-во жалоб превосходит кол-во оценок на ответ, тогда он удаляется и запись снова становится открытой, иначе нет

• Запись с просьбой о помощи можно редактировать в течение 20 секунд, после того как на нее ответят. Также и у того, кто отвечает, он может редактировать свой ответ, пока не истечет время.

Задачи, решенные с помощью Ajax:

1. Обработка запроса с выставлением оценки на ответ пользователя. Оценка сразу отправляется на сервере и без перезагрузки страницы мы видим как изменится рейтинг

2. Обработка жалобы, она сразу отправляется на сервер и также без перезагрузки страницы высвечивается сообщение, что жалоба отправлена

Использованные модули:

Celery, Flask, flask-login, flask-migrate, flask-restful, flask-wtf, json, requests, time.sleep, hashlib, uuid, sqlalchemy.
